# Evaluation Failure Log

Track query quality across test sessions. For each query, note:
- **Defined terms?** Did the response define the key term (genus + differentia) before reasoning?
- **Syllogism?** Was there explicit syllogistic structure (major/minor premise → conclusion)?
- **Cited passages?** Were the provided passages quoted or closely paraphrased?
- **Hedging?** Any "interesting perspective", "some might argue" without resolution, or refusal to take a position?
